About

I'm a student developer based in Multan, Pakistan, currently focused on Python, with C++ and HTML rounding out my foundation. Most of what I build are desktop utilities — a calculator, a contact book, a small library system — the kind of programs that force you to actually finish the boring parts: input validation, saved data, an interface that doesn't break when someone clicks the wrong thing twice.

I like projects small enough to complete end to end, and I've been using that constraint to slowly work up to bigger, multi-file systems.
